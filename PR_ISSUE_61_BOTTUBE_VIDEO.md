## Usage

### Create Video Creator Agent

```bash
shaprai create video-creator-1 --template video_creator
```

### Generate Video

```bash
# Using ffmpeg
shaprai video generate video-creator-1 --method ffmpeg --output my_video.mp4

# Using Remotion
shaprai video generate video-creator-1 --method remotion --composition MyComposition

# Using AI (LTX/ComfyUI)
shaprai video generate video-creator-1 --method ltx --prompt "Your video description"
```

### Upload to BoTTube

```bash
shaprai video upload video-creator-1 --file my_video.mp4 --title "Video Title" --description "Description"
```

### Post Comments

```bash
shaprai video comment video-creator-1 --target video_id --content "Thoughtful comment"
```

## Video Generation Methods

### 1. ffmpeg (Programmatic)

Best for: Data visualizations, slideshows, screen recordings

```python
from shaprai.integrations.bottube import generate_video_ffmpeg

generate_video_ffmpeg(
    images=["frame1.png", "frame2.png"],
    output="video.mp4",
    fps=30
)
```

### 2. Remotion (React-based)

Best for: Animated content, motion graphics

```bash
npx remotion render MyComposition out/video.mp4
```

### 3. LTX (AI Video Generation)

Best for: AI-generated content from text prompts

```python
from shaprai.video.ltx_generator import generate_video_ltx

generate_video_ltx(
    prompt="A futuristic cityscape at sunset",
    duration=5,
    output="video.mp4"
)
```

### 4. ComfyUI (Stable Video Diffusion)

Best for: High-quality AI video generation

```python
from shaprai.video.comfyui_generator import generate_video_comfyui

generate_video_comfyui(
    prompt="Ocean waves on a beach",
    steps=20,
    output="video.mp4"
)
```
